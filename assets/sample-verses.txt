This sample file contains various verse formats that the extraction algorithm can recognize and parse. The system will attempt to identify:
- Book names (Genesis, Exodus, Matthew, etc.)
- Chapter and verse numbers
- The actual verse text
- Context and formatting

The confidence scoring system will rate each extracted verse based on:
- Recognition of biblical language patterns
- Proper verse structure and formatting
- Known Bible book names
- Sentence structure and length

Users can then review the extracted verses, select which ones to import, and create new collections in their SpiritAmmo armory.
